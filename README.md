# alisp

**alisp** is a small Lisp-like programming language written in C, inspired by the [*Build Your Own Lisp*](https://www.buildyourownlisp.com/) guide.

## Build

### Compile

```sh
make # debug
make debug
make release
make clean
```

## Run

### REPL

```sh
./alisp
```

### Run a source file

```sh
./alisp examples/arith.al
```
